Bryg Language Reference
===================================================

*This reference presents the bryg template language for the JVM. The reader of this reference is expected to know Java. This document is not a tutorial, it merely describes the features of the language.*

Bryg is designed and developed by `Marco Pennekamp <http://pennekamp.me/>`_.
